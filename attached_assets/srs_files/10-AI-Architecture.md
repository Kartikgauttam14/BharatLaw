# AI Architecture

## AI features
- Judgment summary
- Key issues
- Important holdings
- Related cases
- Related acts
- Section extraction
- Query rewrite

## AI approach
- Use embeddings for semantic retrieval
- Use LLM only on retrieved legal text
- Keep source citations attached
- Never fabricate citations

## Output rules
- Summaries must cite source document IDs
- AI must show confidence where possible
- AI must not replace legal advice
