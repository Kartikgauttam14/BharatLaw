# Data Pipeline

## Stages
1. Discover source URL
2. Download raw file
3. Extract text
4. Normalize metadata
5. Deduplicate
6. Store raw artifact
7. Store clean text
8. Index for search
9. Generate AI summary
10. Publish through API

## Data quality checks
- Empty text detection
- Broken PDF detection
- Duplicate hash detection
- Metadata completeness
- Language detection
