# Indian Law App SRS Bundle

This bundle is designed for an AI coding agent to build a legal research platform for India.

## Included
- Master SRS
- Architecture documents
- Database design
- API specification
- Scraper strategy
- AI/semantic search plan
- Security, testing, deployment, and roadmap docs

## Intended use
Paste the folder into your coding workflow, or upload the files to Replit AI Agent one by one if needed.

## Core principles
- Use official public sources first
- Avoid copyrighted commercial databases unless licensed
- Store raw source URL and fetch timestamp for every record
- Normalize metadata early
- Keep ingestion, parsing, indexing, and API layers separate
