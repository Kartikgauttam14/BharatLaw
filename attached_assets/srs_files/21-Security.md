# Security

## Threats
- scraping abuse
- injection attacks
- XSS
- broken auth
- data tampering
- brute force login

## Controls
- parameterized queries
- validation
- rate limiting
- security headers
- CSRF protection if cookies are used
- audit logging
- encrypted secrets

## Legal safety
- store source references
- avoid unauthorized redistribution of protected texts
- keep commercial sources behind license checks
